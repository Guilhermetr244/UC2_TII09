Aula Tabelas

introdução as tabelas em

tabelas são estruturas em HTML que permitem a organização de dada em linhas e colunas. são comunte usadas para exibir calendarios, horarios, listas de preços, etc.

Estrutura basica de uma tabela

A estrutura básica de uma tabela em HTML e composta por varias tags:

  • table: Define o inicio e fim de uma tabela.
  • TR: Define uma linha na tabela
  • td: Define uma celula dentro de uma linha (dados)
  • th: define uma celula de cabeçalho

    • Exemplo 1: tabela basica

    nome idade profissão
    ana 28 Engenharia
    bruno 34 medico

    Explicação

    • Nesse Exemplo, Usamos th para o cabeçalho e td para os dados.
    • o tributo border="1" é usado para adicionar bordas a tebela
    Disciplina UN1 UN2 UN3 UN4
    MATEMATICA 8 5 5 5
    INGLES 7 8 8 7

    MESCLAGEM DE CELULAS

    A MESCLAGEM DE CELULAS PODE SER FEITA TANTO EM COLUNAS QUANTO EM LINHAS.

    • COLSPAN: MESCLA CELULAS HORIZONTALMENTE (COLUNAS)
    • ROWSPAN: MESCLA CELULAS VERTICALMENTE (LINHAS)
    NOME IDADE profissão
    carlos 38 professor
    31 arquiteto
    daniela Designer Grafico

    Explicação

    • rowspan-"2": A celula "carlos" se expande VERTICALMENTE para cobrir duas linhas
    produto quantidade preço unitario
    caneta 10 R$ 2,00
    caderno 5 R$ 15,00
    Total: R$ 95,00

    Explicação:

    • thead: Define o cabeçalho da tabela
    • tbody: Define o corpo da tabela, onde ficam os dados
    • tfoot: Define o rodapé, ideal para somatórios ou resumos

    Tabela Aninhada

    É possível colocar uma tabela dentro de outra célula de tabela. Isso é conhecido como aninhamento de tabela.

    Exemplo de Tabela Aninhada:

    Produto Detalhes
    Laptop
    Modelo Preço
    Apple M1 R$ 7.000,00
    Lenovo Ideapad R$ 3.500,00
    caneta azul
